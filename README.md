# NMICRO-CLIENT

## Install

		$ git clone https://github.com/daviddauvergne/nmicro-client.git
		$ cd nmicro-client/
		$ npm install

## Help

		$ node main.js --help

## Start test

		$ node main.js /path/to/project --test

OR

		$ node main.js /path/to/project -t


## Start demo

		$ cd /path/to/nmicro-client/
		$ node main.js ./demo

In your browser: http://localhost:8080
